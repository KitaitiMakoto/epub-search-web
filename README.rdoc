= EPUB Search
Personal search engine for EPUB files.

This projec was retired. Consider {EPUB Sercher}[https://github.com/ranguba/epub-searcher] for the same purpose.

== Requirements
- MySQL
- mroonga
- MeCab
- MeCab adapter for mroonga
- Redis for Resque
- libxml2 and libxslt for Nokogiri
- libzip for Zip/Ruby
